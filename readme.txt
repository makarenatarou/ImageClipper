■ImageClipperについて
　エビデンス、エビデンス、エビデンス、エビデンス、、、うるせーよ！
　某X社がエビデンスは？と聞くのでWindows10の「Win+Shift+s」をもっと活用しやすいプログラムを作りました。

■インストール方法
　0.exeファイルを任意の場所に配置
　1.Install.ps1をPowerShellで実行。

■アンインストール方法
　1.Uninstall.ps1をPowerShellで実行。レジストリが削除されます。残りのexeはそのまま捨ててください。

■使い方
　1.Win+Shift+sでクリップボードにイメージを保存させる。
　2.エクスプローラの背景で右クリック→ImageClipperで画像保存をクリック。
　3.yyyy-MMdd-HHmmss.pngで出力されます。
　エラー処理はしていません。悪しからず。
